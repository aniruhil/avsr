Statistical Methods in Plant Biology (PBIO 3150/5150)

The course syllabus is available here.

Download the latest daily build of RStudio for your operating system from here.

Whitlock & Schluter have provided the data used in the text here. You do not need to but if you have no or spotty internet connection at home you may want to download all the data (provided as a single zip archive) and extract these data to a single folder, maybe to your OU Box account. On Box create a folder for the course and save all your notes, assignments, data, and exams. This will make it easy for you to set your working directory for all coursework.
